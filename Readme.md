# Обработка оценок студентов
## Назанчение
Проект считывает файлы с данными об успеваемости студентов и формирует отчет.

## Подробности
- Отчёт включает в себя
- - нумерацию,
- - список студентов
- - среднюю оценку студента по всем предметам.
- Студенты сортируются по успеваемости.
- Входных файлов может быть несколько,
- название отчета передается в виде параметров --files и --report (см. пример ниже).
- Отчёт формируется по всем переданных файлам, а не по каждому отдельно.

## Пример использования

```shell
python main.py --files students1.csv students2.csv --report file_report
```
