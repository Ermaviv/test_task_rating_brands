# Обработка оценок студентов
## Назначение
Скрипт читает файлы с данными о рейтингах товаров
разных брендов и формирует отчеты. 

## Подробности
- Входные файлы:
    - может быть несколько,
    - представляют из себя csv-файл(ы)
    - передаются после параметра `--files`
- Выходной файл (отчет):
  - передается только одно название,
  - передается после параметра `--report`.
- Отчёт формируется по всем переданных файлам, а не по каждому отдельно.
- Отчёт включает в себя
- - нумерацию,
- - список брендов
- - среднюю оценку всех товаров по каждому бренду.
- Бренды сортируются по рейтингу.

## Пример использования

```shell
python main.py --files products1.csv products2.csv --report average-rating
```
*Указанный пример работает при условии:
- запуск идет из самой директории
- сами файлы `products1.csv` `products2.csv`
  - существуют
  - находятся в директории программы
  - имеют валидные данные
